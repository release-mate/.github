# .github

This repository hosts the organization profile for
[@release-mate](https://github.com/release-mate). GitHub renders
[`profile/README.md`](profile/README.md) as the landing page shown to anyone
visiting the organization.

For information about Release Mate itself — what it does, how it works, and
how to install it — see [`profile/README.md`](profile/README.md).

## Editing the profile

1. Edit [`profile/README.md`](profile/README.md).
2. Open a pull request.
3. Once merged to `main`, GitHub updates the rendered profile within a few
   minutes.
